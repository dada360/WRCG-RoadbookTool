## 2. 项目总体说明
当前项目是一个基于 **HTML + CSS + 原生 JavaScript + Canvas** 的路书时间轴编辑器。  
核心是“距离时间轴”与“标记编辑”，在交互上强调类似剪辑软件的流畅拖拽、缩放和定位体验。

---

## 3. 模块划分

### 3.1 页面结构模块（HTML）
文件：`index.html`

主要职责：
- 顶部创建区：路书名称、起始距离、创建按钮
- 状态面板：路书名称、起始距离、当前距离、状态、可视范围、选中标记位置
- 时间轴区域：
  - 工具栏（导入/导出）
  - 时间轴 Canvas
  - 左右刻度文本
- 右键圆盘菜单容器（Canvas）
- 自定义通知栏容器

---

### 3.2 样式模块（CSS）
#### `css/base.css`
- 全局颜色变量、字体、按钮、输入框基础样式
- 通知栏样式（顶部弹出、自动淡出）

#### `css/layout.css`
- 页面区块布局（top-bar / status-panel / timeline-section）
- 响应式布局规则

#### `css/timeline.css`
- 时间轴视图、Canvas 尺寸、交互态光标样式
- 时间轴提示信息样式

#### `css/context-menu.css`
- 右键圆盘菜单容器与动效样式

---

### 3.3 配置与工具模块（JS）
#### `js/config.js`
- 统一配置项：
  - 标记类型集合 `MARKER_TYPES`
  - 播放速度 `PLAY_SPEED_MPS`
  - 坐标边距 `AXIS_PADDING`
  - 浮点比较容差 `EPSILON`

#### `js/utils.js`
- 通用工具函数：
  - `toOneDecimal(value)`：保留一位小数
  - `formatDistance(value)`：格式化为 `xx.xm`
  - `clamp(value, min, max)`：数值夹紧

#### `js/state.js`
- 全局状态容器：
  - 路书信息（名称、起始距离、当前距离）
  - 标记数据与选择状态
  - 缩放、平移、拖拽、播放等交互状态

#### `js/dom.js`
- 所有 DOM 引用集中管理，避免散落 `getElementById` 调用

---

### 3.4 核心业务模块（JS）
文件：`js/app.js`

主要职责：
- 时间轴 Canvas 渲染
- 刻度与指针绘制
- 标记绘制、命中检测、拖拽移动
- 播放循环（空格播放/暂停）
- 缩放、平移、键盘控制
- 右键长按圆盘菜单（Canvas）
- 导入/导出 JSON
- 自定义通知栏提示
- 标记吸附与避让排布

## 4. 核心功能清单

### 4.1 路书创建
- 必填路书名称
- 输入起始距离（>0）
- 创建后初始化状态：
  - `currentDistance = startDistance`
  - 清空标记
  - 重置视图与交互状态

### 4.2 时间轴交互
- 方向：左大右小
- 滚轮缩放（以鼠标锚点缩放）
- 中键平移
- 刻度/角标拖动调整指针
- 左右方向键微调（含长按连续）

### 4.3 播放控制
- 仅保留 `Space` 控制播放/暂停
- 播放时距离递减
- 到 `0.0m` 自动停止

### 4.4 标记创建与编辑
- 右键长按 0.1s 打开圆盘菜单
- 松开右键确认创建标记
- 标记支持点击选中、拖拽移动、Delete 删除
- 状态栏实时显示选中标记位置

### 4.5 标记视觉与可读性
- 样式：左直右圆角矩形标签
- 标签长度随内容自适应
- 多标签近距离时自动分层避让（上下交替）
- 标签显示排序 ID（如 `#01 右3`）

### 4.6 吸附机制
拖动标记时支持吸附到：
1. 当前指针位置
2. 其它标记尾部

并支持持续关联，避免缩放后吸附关系失效导致重叠。

### 4.7 右键圆盘菜单
- Canvas 绘制甜甜圈菜单
- 一级/二级菜单结构
- 二级打开后隐藏其他一级
- 文本沿弧布局以提升识别性
- 中心区域用于关闭二级层级

### 4.8 导入导出（JSON）
- 导出字段：
  - `version`
  - `roadbookName`
  - `startDistance`
  - `currentDistance`
  - `markers[]`（含 `id`、`distance`、`type`）
- 导出 `markers` 按时间线顺序输出，ID 格式为 `#01/#02/...`
- 导入时执行合法性校验与数值夹紧

### 4.9 自定义通知栏
- 替代 `alert`
- 从顶部弹出
- 2 秒自动关闭
- 支持 success / error 视觉区分
